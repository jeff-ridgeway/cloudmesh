create_cluster(name, names, ips, management)[source]¶

creates a cluster with the given name specification ip specifications, and the identification of management node.

Parameters:
  • name – name of the cluster
  • names – the names of the cluster servers. ‘i[001-003]’. creates the objects with names i001, i002, i003
  • ips – the names of the ips for the servers. ‘i[001-003].futuregrid.org’ creates the ips for the previously defined names
  • management – the names of the management nodes. ‘i[001-002]’ sets the nodes i001 and i002 to management nodes. The rest will be set to compute nodes automatically.
