1   About this document¶

This document will continualy evolve to reflect the newest findings in our agile programming approach. The document is hosted on github at

  • https://github.com/cloudmesh/plan

You can check it out with:

git clone git@github.com:cloudmesh/plan.git

Once you make changes you can say:

make sphinx

Make sure you do not have any errors in the document before pushing it to gh-pages. The compiled documentation is generated while using the command:

make gh-pages
