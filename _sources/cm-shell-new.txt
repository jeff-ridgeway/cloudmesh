Label
----------------------------------------------------------------------

The `label` command allows a user to set the defualt values for the prefix and the id that are used for naming the virtual machines. An example is::

  cm> label --prefix=gregor --id=7 --width=3

This will set the label for the next vm id to `gregor-007`. The width parameter specifies the width of the number that is filled with `0`.
